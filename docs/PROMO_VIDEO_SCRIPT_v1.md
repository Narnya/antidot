## Цель ролика

Объяснить за 20–25 секунд, что Антидот — это **не дейтинг и не календарь мероприятий**, а способ безопасно знакомиться через реальные офлайн-встречи.

**Один месседж:** «Сначала — общее занятие. Потом — знакомство.»

**CTA:** запомнить имя `Antidot` и зайти на `antidot.space` (закрытая бета).

| Параметр | Значение |
|----------|----------|
| Длительность | 22 сек (target), приемлемо 20–25 сек |
| Аспект | 9:16 (1080×1920) — основной, для Reels/Shorts/TikTok |
| Альтернатива | 1:1 (1080×1080) — для feed; 16:9 (1920×1080) — для лендинга |
| Тип | Скринкаст Figma-прототипа + текстовые оверлеи + лёгкий motion + войсовер |

| Палитра | Warm minimal (см. memory: figma-prototype-build) |

## Запреты (по safety-инвариантам Product Core)

В кадре **НЕ должно появиться**:

1. Точная локация события (адрес, пин на карте) — Инв. 1, 9.
2. Открытый чат / DM с незнакомым пользователем — Инв. 2.
3. Число trust score (`Trust: 74/100` и т.п.) — Инв. 3. Допустимы только мягкие бейджи (Verified, Reliable attendee и т.п.).
4. Любая dating-механика: свайпы, мэтчи, сердечки, проценты совместимости, карусели лиц.
5. Массовые мероприятия, ночные клубы, party-vibe.
6. Слова `dating`, `match`, `свайп`, «найди свою половинку», «знакомства» в романтическом контексте.

Если редактор/монтажёр предложит что-то из этого — отказать со ссылкой на Product Core.

## Войсовер — 3 варианта тона

Выбрать один; остальные — запасные/для A/B.

### Вариант A — тёплый, человечный (рекомендуется)

> Знакомиться онлайн стало странно. А что, если начать с общего занятия? Подаёшься. Хост одобряет. Приходишь — в реальный мир. Без свайпов. Без сообщений случайным людям. **Антидот. Скоро.**

*(~22 сек при темпе ~2.6 слова/сек)*

### Вариант B — спокойный, чуть медитативный

> Раньше люди знакомились в реальности. Потом — через экран. Антидот возвращает первое. Маленькие встречи. Знакомые лица. Доверие — по умолчанию. **Антидот. Antidot.space.**

### Вариант C — короткий, чуть ироничный (для молодой аудитории)

> Свайпы устали. Чаты с незнакомыми — тоже. Антидот предлагает простое: сначала встреться, потом общайся. **Antidot.**

**Голос (ElevenLabs / аналог):**
- Русский, ~25–35 лет, тёплый тембр, не «дикторский».
- Темп: спокойный, ~2.5–2.8 слова/сек, без энергичных «продающих» интонаций.
- Подходящие голоса в ElevenLabs (на проверку): `Charlotte`, `Daria`, либо кастомный клон спокойного голоса.

## Текстовые оверлеи — точные строки

Для монтажа, в порядке появления:

1. `Знакомиться онлайн стало странно.`
2. `А что, если — начать с общего занятия?`
3. `Подайся. Получи одобрение.`
4. `И — встречаешься в реальности.`
5. `Без свайпов. Без сообщений случайным людям.`
6. `Antidot` (логотип) + `antidot.space` · `Скоро. Закрытая бета.`

**Типографика:** один шрифт из дизайн-системы (см. figma-prototype-build). Кегль крупный, межстрочный воздух, выключка по левому краю. Никаких теней/градиентов/«кинонадписей».

## Аудио

- **Музыка:** lo-fi acoustic / warm piano / soft ambient. BPM 70–90. Без EDM, без drops.
- **Источники (бесплатные/доступные):** YouTube Audio Library, Pixabay Music, Uppbeat (free tier), Epidemic Sound (по подписке).
- **Поисковые запросы:** `warm acoustic loop`, `soft piano cinematic`, `cozy lo-fi`, `ambient warm`.
- **Микс:** музыка −18 dB под войсовер, в outro поднять до −12 dB.
- **UI-звуки:** один лёгкий «тап» на кадре №3 (Apply). Больше не нужно.

## Сборка в Jitter — recommended workflow

> **Что такое Jitter:** motion-design инструмент в браузере, заточенный под анимацию готовых дизайнов из Figma. Бесплатный tier хватает на ролик такого масштаба. Сайт: [jitter.video](https://jitter.video).
>
> **Почему именно Jitter, а не AI-видео (Sora/Runway/Veo/Kling):** AI-модели не воспроизведут UI «Антидот» (карточки событий, бейджи, кнопки) — текст и иконки получатся невнятным месивом, бренд потеряется. AI допустим только на Кадр 04 (фото без интерфейса).

### Quickstart

1. **Импорт из Figma (preferred):** в Jitter установить Figma-плагин, открыть файл прототипа `xZkaKij7DhLPpRE3Ob0Znd`, выделить 6 phone-фреймов (`Kadr 01–06`, ID см. ниже) и нажать «Import». Слои сохранятся — можно анимировать каждый объект отдельно.
2. **Фоллбэк (без плагина):** импортировать готовые 1080×1920 PNG из [`promo-assets/jitter-export/`](../promo-assets/jitter-export/). В этом случае анимировать можно только кадр целиком (целиком фейдить / сдвигать / зумить).
3. Установить **canvas size 1080×1920, 30 fps, длительность 23 сек**.
4. Расставить кадры на таймлайне по таблице ниже.
5. Применить пресеты Motion timing (см. ниже).
6. Экспорт MP4 (H.264, 8–12 Mbps).
7. Сборка с войсовером + музыкой в CapCut/DaVinci.

## Motion timing — Jitter пресеты

Все длительности ниже — таймкоды внутри кадра, не от начала ролика. Между кадрами — **cross-fade 200 мс**, чтобы стыки не били глаз.

**Глобальный easing:** `Out Cubic` для входа объектов, `In-Out Cubic` для движений камеры, `Linear` только для breath/Ken Burns. Никаких bounce/elastic — это под детский тон, не под Антидот.

### Кадр 01 — Хук · 4s · 0:00–0:04

| Слой | Пресет | Когда | Длит |
|------|--------|-------|------|
| Тёмный фон phone | `Fade In` | 0.0s | 0.4s |
| Текст «Знакомиться онлайн стало странно.» | `Fade In` + `Slide Up 20px` | 0.6s | 0.6s |
| Коралловая точка | `Scale In 0→1` (Out Cubic) | 1.2s | 0.4s |
| (всё) | hold | 1.6–3.4s | — |
| (всё) | `Fade Out` | 3.4s | 0.6s |

**Звук:** мягкий piano-аккорд на 0.6s.
